# Roadmap

This file tracks what I working on right now and what is coming next for OpenMirror. Updates are made step by step.

## In progress

- **Face training and camera improvements**
  - More reliable face descriptor capture in the setup dashboard.
  - Better camera handling on Raspberry Pi and in the browser.
  - Improved accuracy and speed of face recognition.
  - Privacy-first local storage remains the default.

## Near term

- Stability improvements across core services and built-in modules.
- Setup experience polish, clearer error messages, and smoother first install.
- Module quality: testing and refining newer modules such as Immich Frame, Spotify, Todoist, and Travel Time.
- Hardware support expansion, easier sensor configuration, and more test devices.

## Longer term

- Pre-flashed SD card images and easier Raspberry Pi deployment.
- Cloud/SaaS remote management features.
- More built-in modules based on real-world usage.
- Better documentation.

## How I work

OpenMirror is built in small, focused steps. Each improvement is tested, documented, and released before moving to the next one. If you want to influence the roadmap, open an issue or discussion on GitHub.
